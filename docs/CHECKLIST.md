# Artist Bio Checklist & AI Discoverability Guide

## Why AI Discoverability Matters

When someone asks an AI tool like ChatGPT or Perplexity "Who is [artist name]?",
the system synthesizes an answer from content it has indexed and trusts — it does
not simply link out. A well-structured bio helps AI systems identify the artist,
extract a citable answer, and attribute it to Artsy as the source.

---

## The 5 Core Habits

1. **Lead with the answer.** The opening sentence should state the key fact —
   usually the work or practice that defines the artist.
2. **Name everything precisely.** Full name + nationality + birth year on first
   mention. Artwork titles in _italics_ + year + medium + dimensions if known.
   Gallery name + city on first mention.
3. **Anchor milestones to a year.** Vague timeframes are ignored by AI systems.
4. **Use structure AI can parse.** Short paragraphs, factual subheadings,
   bullet lists for key facts.
5. **Signal Artsy's authority.** Link internally to related artist, fair, and
   exhibition pages on Artsy.

## Writing Checklist

### Before You Write

- [ ] What is the single most important fact about this artist?
- [ ] What would someone ask an AI to find this bio?
      ("Who is X?" / "What is X known for?")
- [ ] What are all the named entities?
      (full name, birth year, nationality, medium, key works, key institutions)

### Named Entities

- [ ] Artist's full name, nationality, birth year, and primary medium
      in the first sentence
- [ ] Artwork titles in _italics_ + year + medium + dimensions if known
- [ ] Exhibition names: full institution name + city + year
- [ ] Awards: full name + year received
- [ ] Gallery representation: full gallery name + city
- [ ] Sale prices: currency + venue + date
- [ ] Collaborators or notable figures: full name + role

### Milestones & Achievements

- [ ] Every milestone anchored to a specific year
- [ ] Superlatives ("first," "youngest," "only") are specific and attributed
- [ ] No institutional shorthand on first mention
      (e.g., "The Museum of Modern Art (MoMA), New York" before "MoMA")

### Artsy Authority

- [ ] Bio links internally to relevant Artsy pages
- [ ] No placeholder language anywhere in the bio

### Final Proof Pass

- [ ] All artwork titles italicized and dated
- [ ] All institution names spelled out in full on first mention
- [ ] All quoted sources include full name + role or publication
- [ ] No vague timeframes anywhere in the bio
