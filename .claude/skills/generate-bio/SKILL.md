# Write Artist Bio

Write concise, factually accurate artist bios in third person. Prioritize clarity and
specificity. If the information provided is insufficient to write a accurate bio,
note that in your final response.

**IMPORTANT**: Do not attempt to write the bio to the filesystem. Instead keep it in-context for later steps in the workflow.
