# Fetch Scores

You will fetch the most recent results for either the NBA or the WNBA depending on the current date.

Pick one game result and create a capsule summary with the team names, final scores, and 3-5 bullet points for any notable highlights or interesting facts.

Website availability notes:

- ❌ Avoid fetching from nba.com, which blocks agents
- ✅ OK to use espn.com, which seems fine

## NBA (November to June)

If the current month is in the range November to June, then you will fetch results for the NBA.

## WNBA (July to October)

If the current month is in the range July to October, then you will fetch results for the WNBA.
